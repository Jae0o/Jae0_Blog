# 블로그 만들기 프로젝트!

### Commit Conventions

| Emoji 😀         | Description                                         |
| ---------------- | --------------------------------------------------- |
| [Feat ✏️]        | 새로운 기능 추가                                    |
| [File 📝]        | 파일을 생성할 때                                    |
| [Struct 🚜📝]    | 파일 구조를 변경할 때 📝과 함께 사용                |
| [Design 🎨]      | UI / style 개선시                                   |
| [Rename 🏷️]      | 이름 변경 위주의 작업을 진행 할 때                  |
| [Remove 🗑️]      | 코드 또는 파일 제거할 때 , @CHANGED주석 태그와 함께 |
| [Improve 🐎]     | 성능을 향상시킬 때                                  |
| [Refactoring ⚙️] | 코드를 리팩토링 할 때                               |
| [Bug 🐛]         | 버그 reporting할 때, @FIXME 주석 태그 삽입          |
| [Fix 🪛]         | 버그를 고칠 때                                      |
| [Release 💎]     | New Release                                         |
| [Version 🔖]     | 버전 태그                                           |
| [Write 📚]       | 문서를 쓸 때                                        |
